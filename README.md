# ContributorsList
Sample Android app which displays list of all contributors of [ButterKnife](https://github.com/JakeWharton/butterknife) Github project

## Technical deatils and used libs
* Works in portrait and landscape
* MinSdk 19
* AppCompat
* RecyclerView
* CardView
* Retrofit 2
* RxJava 2
* Picasso
